## Random.color

* Random.color()

随机生成一个有吸引力的颜色，格式为 '#RRGGBB'。

```js
Random.color()
// => "#3538B2"
```

## Random.hex

* Random.hex()

随机生成一个有吸引力的颜色，格式为 '#RRGGBB'。

```js
Random.hex()
// => "#3538B2"
```

## Random.rgb

* Random.rgb()

随机生成一个有吸引力的颜色，格式为 'rgb(r, g, b)'。

```js
Random.rgb()
// => "rgb(242, 198, 121)"
```

## Random.rgba

* Random.rgba()

随机生成一个有吸引力的颜色，格式为 'rgba(r, g, b, a)'。

```js
Random.rgba()
// => "rgba(242, 198, 121, 0.13)"
```

## Random.hsl

* Random.hsl()

随机生成一个有吸引力的颜色，格式为 'hsl(h, s, l)'。

```js
Random.hsl()
// => "hsl(345, 82, 71)"
```
